# Oscar The Knight

A self-contained hand-painted 2D platformer prototype built with HTML, CSS, and Canvas.

## Run Locally

```bash
npm start
```

Then open `http://localhost:4173`.

## Controls

- Move: arrow keys or A/D
- Jump: Space, W, or Up
- Strike: J, K, or X
- Mobile: on-screen controls

## Deploy

```bash
npm run deploy:vercel
```
