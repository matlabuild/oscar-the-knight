# Oscar The Knight Asset Notes

`assets/generated/oscar-painted-atlas.png` was generated with the built-in image generation tool, then converted from a solid green chroma-key background to alpha using the local `remove_chroma_key.py` helper.

The following level backgrounds were also generated with the built-in image generation tool and copied into the project:

- `assets/generated/bg-moonlit-gate.png`
- `assets/generated/bg-briar-quarry.png`
- `assets/generated/bg-ember-keep.png`
- `assets/generated/bg-starfall-observatory.png`
- `assets/generated/oscar-environment-atlas.png`

Prompt summary:

```text
Create a polished sprite sheet for a storybook medieval platformer called Oscar The Knight. Include Oscar, a small brave knight with cream steel armor, gold visor, red plume, burgundy cloak, round golden shield, and bright sword; also include a thorn imp, a bat, an ember spirit, a thorn regent mini-boss, a moon shard collectible, a crest relic, and a lantern relic. Hand-painted 2D game sprites, crisp readable silhouettes, premium indie platformer look, soft painterly highlights, clean edges, no pixel art, no 3D render. Organized sprite atlas on a flat #00ff00 chroma-key background.
```

Background prompt summary:

```text
Create full-screen 16:9 hand-painted 2D platformer backdrops for Moonlit Gate, Briar Quarry, Ember Keep, and Starfall Observatory. Premium storybook indie platformer art, parallax-friendly wide landscapes, no characters, no UI, no text, lower third kept readable for gameplay.
```

Environment atlas prompt summary:

```text
Create a premium hand-painted environment sprite sheet for Oscar The Knight with modular mossy stone platforms, carved observatory platforms, ember keep lava-stone platforms, wooden quarry platforms, ornate arched doors/gates, checkpoint banner pole, thorn brambles, spikes, lava vents, crates, hanging lanterns, brass chains, vines, flowers, and foreground rocks. Organized on a flat #00ff00 chroma-key background for alpha removal.
```
